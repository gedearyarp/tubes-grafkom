## Fitur Program

1. Menggambar garis dan bentuk (persegi, persegi panjang, poligon)
2. Memindahkan objek
3. Mewarnai objek
4. Mewarnai titik sudut pada bidang
5. Menambah dan menghapus titik sudut pada poligon
6. Membuat convex hull dari poligon
